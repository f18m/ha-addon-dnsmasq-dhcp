DHCP Status Summary

Notes:

  • Several columns like Friendly Name, Description, Link are populated using the per-DHCP-client metadata provided in the dhcp_client_settings app configuration. Such contents are not exposed over DHCP/DNS and are only meant to be used as a way to provide more human-friendly information about the clients in this UI.
  • The IP Address column contains the currently allocated IP address. Note that the address might be located outside the DHCP range in the case the DHCP client has an IP reservation assigned via the dhcp_client_settings app configuration, configured to be outside the DHCP range.
  • The Expires in column contains the count down to the next DHCP lease renewal formatted as HH:MM:SS.

Notes:

  • All clients last seen more than {{ .DHCPForgetPastClientsAfter }} ago are automatically erased and do not appear in this table. This is customizable via the forget_past_clients_after setting.
